# WDI\_TLV\_P2P\_ADVERTISED\_SERVICES


WDI\_TLV\_P2P\_ADVERTISED\_SERVICES is a TLV that contains a list of advertised services.

## TLV Type


0xEF

## Length


The sum (in bytes) of the sizes of all contained TLVs.
