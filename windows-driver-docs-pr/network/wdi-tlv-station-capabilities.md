# WDI\_TLV\_STATION\_CAPABILITIES


WDI\_TLV\_STATION\_CAPABILITIES is a TLV that contains the capabilities of a station.

## TLV Type


0x11

## Length


The sum (in bytes) of the sizes of all contained elements.
