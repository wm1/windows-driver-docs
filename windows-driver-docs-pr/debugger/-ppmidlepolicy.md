# !ppmidlepolicy

The **!ppmidlepolicy** command displays C-state policy.

```
!ppmidlepolicy
```

## <span id="Parameters"></span><span id="parameters"></span><span id="PARAMETERS"></span>Parameters

None

This command is supported on the following versions of Windows:

- Windows 7
